  System Requirements
  -------------------

  JDK:
    1.7 or above (this is to execute Maven - it still allows you to build against 1.3
    and prior JDK's).
  Memory:
    No minimum requirement.
  Disk:
    Approximately 10MB is required for the Maven installation itself. In addition to
    that, additional disk space will be used for your local Maven repository. The size
    of your local repository will vary depending on usage but expect at least 500MB.
  Operating System:
    Windows:
      Windows 2000 or above.
    Unix based systems (Linux, Solaris and Mac OS X) and others:
      No minimum requirement.

  - SpreadSheet.zip
  ----------------
    
		1) Unpack the archive where you would like to store the project, e.g.:

		Unix-based operating systems (Linux, Solaris and Mac OS X)
		  tar zxvf SpreadSheet.zip
		Windows
		  unzip SpreadSheet.zip

		2) A directory called "SpreadSheet" will be created.

		3) Run Apache NetBeans IDE X.X

		4) Run Main.java and follow the Console instructions.
			You are able to create a new spreadsheet or load the spreadsheet.txt already created for you.
		
		Program features
		-----------------
		
		- spreadsheet.txt
		-----------------
		This file is already created with a lot of combinations already implemented:
			- Num + Num ;
			- CellReference*Num ; 
			- CellReference + Function(Num,Num);
			- Function(CellReference,Num) + Function(CellReference,Num);
			- Function(CellReference:CellReference,CellReference) + Function (CellReference:CellReference,Num);
			- Function(Function(CellReference:CellReference,Num),CellReference);
		
		All the above has been tested with several functions & combinations(which are already set in spreadsheet.txt) 
		and trying to solve any possible Excepction than can happen, any missunderstood function and any failures 
		of the system:
			- CellOutOfRange.
			- Range without being part of function.
			- DivisionByZero.
			- An operation with text.
			- RangeOutOfRange.
			- StackOverFlowException.
			- FileNotFoundException.
			- IOException.
